# Tic Tac Toe

## Overview
The second project in the course was to program a game using the Java Swing API. I chose to make Tic Tac Toe. It is nothing flashy as I only had two weeks to put it together. 
## Purpose
The purpose of this project was to utilize Java Swing to create GUI based programs. 
## Usage
After running, A simple game of Tic Tac Toe starts. Best played with friends.
